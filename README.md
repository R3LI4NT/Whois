# Whois
Herramienta para escanear sitios web que contengan dominios .COM, .NET y .EDU

###### Requerimiento/s
```
Python3
• sudo apt-get install python3
```

## Instalación:
```
➤ git clone https://github.com/r3li4nt/whois
➤ python3 whois.py
```
